🚀 QUICK START GUIDE
==================

Choose your preferred method to start both servers:

1️⃣ DOUBLE-CLICK: start.bat (Windows)
2️⃣ TERMINAL: npm run dev
3️⃣ VS CODE: Ctrl+Shift+P → "Tasks: Run Task" → "🚀 Start Both Servers"

URLs after starting:
🌐 Backend API: http://localhost:3000
📱 Frontend: http://localhost:8081

For physical device testing:
📝 Edit app/services/api.ts
🔄 Change localhost to 192.168.1.159
